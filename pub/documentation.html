Getting started

Notify is a library that can notate and play music easily on the web. To use the library, include the notify.css file in your html and also Notify.js as a script.

Initializing a staff

Here is an example on how to initialize a staff.

const staff1 = new Notify();
const staffEl = staff1.createStaff("TREBLE", "QUARTER", 4);

staffEl contains the html element of the staff. You can now put it in your application through .appendChild(staffEl)

Adding/Changing/Deleting Notes

Once you create a staff, it will by default have one bar in the staff. Empty bars will be filled with rest notes. To "add" a note, you can modify the rest note through this API:

staff1.changeNoteTimeValue(0, 0, "QUARTER");
staff1.changeNotePitch(0, 0, "G4");
staff1.changeNotePitch(0, 0, null);

"Deleting notes" would just be changing the note's pitch value to null

Adding/removing bars

You can add and remove a bar from the end of the staff like this:

staff1.addBar();
staff1.removeBar();

Getting data

To get data on the notes that have been added to the staff, use the following API:

staff1.getAllNotes();
staff1.getSelectedNote();

getSelectedNote will return the data of the currently selected note on the staff and getAllNotes will return the data of all of the notes.

Playing Sound

To play the sound of a note/bar/staff, use the following API:

staff1.playNote();
staff1.playBar(0);
staff1.playStaff(0);

createStaff

Create a staff with clef, beat duration (QUARTER, HALF), and beat count (number of beats in a bar)

createStaff(clef: string, noteDuration: string, noteCount: number, editable: boolean): element
Parameters
clef (string) The clef that you want to use in all capital letters for the staff. Note that library currently only supports the treble clef "TREBLE"
noteDuration (string) The duration of each beat of a bar. Must be all uppercase. ("QUARTER", "EIGHTH", "HALF")
noteCount (number) The number of beats per bar. Note that the library currently supports [1, 2, 4, 8, 9, 12]
editable (boolean = true) Show the UI that edits the staff. It is by default set to true.
Returns
element: Returns an HTML element that contains the staff and the UI.

getStaffElement

Get the main DOM element of this staff

getStaffElement(): element
Returns
element: Returns an HTML element that contains the staff and the UI.

getSelectedNote

Get the duration and pitch of the currently selected note on the staff.

getSelectedNote(): [String, String]
Returns
[String, String]: Returns an array of the duration and pitch of the note.

getAllNotes

Get the duration and pitches of all the notes in the staff.

getAllNotes(): []
Returns
[]: Returns an array of arrays, that indicate each bar, that contains the duration and pitches of each note.

getStaffObject

Get the staff object that contains information about the staff, bars, and notes

getStaffObject(): Staff
Returns
Staff: The javascript Staff object that represents the staff.

changeNotePitch

Change pitch of the note at barIndex, noteIndex to newPitch

changeNotePitch(barIndex: number, noteIndex: number, newPitch: string, responsive: any)
Parameters
barIndex (number) The index of the bar (starting from 0) that contains the note.
noteIndex (number) The index of the note (starting from 0) in the bar.
newPitch (string) The new pitch string of the note in form "{note pitch}{note octave}" As an example "G5" represents the G note in the 5th octave.
responsive (any = false)

changeNoteTimeValue

Change time value of the note at barIndex, noteIndex to newValue

changeNoteTimeValue(barIndex: number, noteIndex: number, newValue: string)
Parameters
barIndex (number) The index of the bar (starting from 0) that contains the note.
noteIndex (number) The index of the note (starting from 0) in the bar.
newValue (string) The new time value string of the note in all capital letters. As an example "QUARTER", "EIGHTH"

changePlayBackSpeed

Change the play back speed in BPM per quarter note.

changePlayBackSpeed(newSpeed: number)
Parameters
newSpeed (number) The BPM (of a quarter note). The higher the number, the fast the playback

changeTimeSignature

Change the time signature of the staff. Warning: Deletes all of the note in the staff.

changeTimeSignature(noteDuration: string, noteCount: number)
Parameters
noteDuration (string) The duration of each beat of a bar. Must be all uppercase. ("QUARTER", "EIGHTH", "HALF")
noteCount (number) The number of beats per bar. Note that the library currently supports [1, 2, 4, 8, 9, 12] playback
